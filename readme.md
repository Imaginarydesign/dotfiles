## Dotfiles

My dotfiles

### Cloning Our Dotfiles To Another Machine

1. git clone git://github.com/Imaginarydesign/dotfiles.git
2. cd ~/dotfiles
3. chmod +x install.sh
4. ./install.sh
5. git submodule init
6. git submodule update
